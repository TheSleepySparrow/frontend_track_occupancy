# Frontend-часть приложения для отслеживания загруженности аудиторий

## Предусловие

```bash  
docker network create tyk-network
```   
## Запуск
```bash
docker compose up --build
```
## Доступ
### Приложение доступно по адресу:
```bash
http://localhost:80
```
## Остановка
```bash
docker-compose down
```
